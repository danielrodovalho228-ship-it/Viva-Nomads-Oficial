# Imagens do site — onde colocar cada arquivo

Suba suas imagens aqui (pelo GitHub: **Add file → Upload files**, arrastando da sua pasta
`viva-nomads-images`). Use **exatamente** estes nomes — assim que estiverem no repositório,
basta avisar que o time troca as fotos interinas (Unsplash) pelos seus arquivos.

Formato ideal: **.webp** (ou .jpg/.png). Veja tamanhos e prompts em `ASSETS.md`.
